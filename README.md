##  How to Play  
1. Run the program. 
2. Enter a **valid 5-letter word**.  
3. Get color-coded hints after each guess.  
4. You have **5 attempts** to guess the correct word.  
5. If you guess correctly, you win! Otherwise, the correct word is revealed.
